# E1 — Synthetic Population Construction

## Purpose

Build the 100,000-agent worker population used by every subsequent simulation experiment. The population is **survey-moment-anchored**: each heterogeneity dimension is drawn from a hand-tuned parametric family whose moments are informed by SCE empirical statistics. The same population is loaded by every fix6.x run.

## Source Files

- Generation script: `Phase2_Code/population_init_engine.py` (386 lines; entry point at line 380)
- Population file: `Phase2_Output/population_v1.npz` (loaded by `Simulation.__init__` in every variant run)
- Schema map: `Phase2_Output/matrix_schema_map.json`
- Moment extraction: `Phase2_Code/extract_distributions.py`
- Anchoring documentation: `撰写版本2/cheacklist/SCE_Question_Wording.md`, `撰写版本2/cheacklist/ABM_P1_Resolution_Report.md` §4

## Protocol

1. The script is executed **once** with `seed=42` to draw 100,000 agents (line 380).
2. Each of the 6 heterogeneity dimensions is generated by a small parametric family (Normal, Lognormal, Exponential, Beta, or discrete categorical) whose target moments are informed by the SCE moment table extracted in Phase 1.
3. Resulting state matrix is saved to `Phase2_Output/population_v1.npz`.
4. Every variant in fix6.1–fix6.5 loads this single file at simulation start; no run regenerates the population.

## Main Outputs

- `tables/E01_T01_Population_Summary_v01.csv` — single-row population provenance summary
- `tables/E01_T02_Population_Diagnostics_v01.csv` — dimension-by-dimension family + anchor + status
- `figures/E01_F01_Population_Marginals_v01.png` — six-panel synthetic-population marginals vs anchoring targets (age / education / employment / liquidity / housing / consumption)

## Key Numbers

- N = 100,000 agents (fixed)
- Initialisation seed = 42
- Heterogeneity dimensions = 6 (search, labour-fragility, liquidity, housing, income-expect, consumption-rule)
- Direct SCE row sampling: **No**
- Survey weights applied: **No**

## Caveats

- The population is **parametric-informed-by-moments**, not resampled from SCE microdata. Wording in the paper must use "survey-moment-anchored" and avoid "calibrated to SCE microdata" or "survey-sampled".
- SCE moments are **unweighted**. The population reflects the unweighted sample distribution, not the U.S. household universe.
- Two dimensions (mobility_friction, flexibility_index) are synthetic constructs whose moments are informed by SCE-implied ranges but are not direct SCE variables.
