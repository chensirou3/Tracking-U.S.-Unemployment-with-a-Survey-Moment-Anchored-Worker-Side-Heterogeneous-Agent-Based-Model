## 二、逐维度审计详情

### 异质性 1：Income Growth Expectation（MVP）

**审计结论：保留，无需修改。**

- **名称准确性**：✅ 准确。刻画的是家庭收入增长预期的水平和不确定性。
- **与主目标一致性**：✅ 直接相关。收入预期影响：(a) 求职强度——乐观者搜索更宽松；(b) 保留工资——高预期者保留工资更高；(c) 劳动参与——极度悲观者可能退出劳动市场。
- **机制解释充分性**：✅ 充分。消费平滑、预防性储蓄、借贷决策、劳动供给四条通道均有理论支持。
- **所属类别**：✅ 预期（Expectation）。
- **题项支持**：✅ Q24系列（密度预测）提供完整的中位数+不确定性信息；Q25v2/Q26v2提供方向性验证。
- **与其他维度重叠**：⚠️ 与Labor Fragility有交互（失业风险影响收入预期），但不是重叠——两者测量不同机制（一个是收入增长信念，一个是就业风险暴露）。
- **分布构造**：✅ Q24_cent50连续分布 + Q24_iqr不确定性，合理。
- **时间属性**：✅ 快速预期变量，月度更新，正确。

### 异质性 2：Labor Fragility（MVP）

**审计结论：保留，需补全正文结构。**

- **名称准确性**：✅ 准确。
- **与主目标一致性**：✅ 最直接相关。Q13new（失业概率）和Q22new（再就业概率）直接构成失业率的微观基础。
- **机制解释充分性**：✅ 充分。
- **所属类别**：✅ 状态+预期。
- **题项支持**：✅ Core Q13new + Q22new 是主干；LM ec14 + l8 是增强。
- **已知问题**：
  - 正文缺失第7-10节，需在字典v2中补全
  - Q22new需反转（1-Q22new）后再与Q13new组合，否则PCA方向可能混乱
- **分布构造**：✅ PCA → Fragility Index 合理，但需注意变量方向一致性。
- **时间属性**：✅ 中速状态变量。

### 异质性 3：Liquidity Fragility（MVP）

**审计结论：保留，需调整数据主干。**

- **名称准确性**：✅ 准确。
- **与主目标一致性**：✅ 直接相关。流动性不足 → 无法承担长期搜索 → 被迫接受低质量工作或退出劳动市场。
- **机制解释充分性**：✅ 充分。
- **已知问题**：
  - 原方案将 HH Finance c1系列和 Informal Work Q6 列为主支持题，但：
    - HH Finance 已停更（2014-2019），不能作为动态主干
    - Informal Work 来自不同调查（波士顿联储），不能个体级匹配
  - **修正**：主干必须改为 Core Q32 + Q33 + Q47（月度可用），HH Finance 和 Informal Work 仅作为分布校准参考
- **分布构造**：✅ 规则分类（H2M / Adequate Buffer / Wealthy）合理，参考 Kaplan-Violante (2014)。
- **时间属性**：✅ 中速状态变量，但负面冲击可快速恶化。

### 异质性 4：Labor Search Friction（MVP，已从第二阶段提升）

**审计结论：保留，层级提升合理。**

- **名称准确性**：✅ 准确。
- **与主目标一致性**：✅ 最直接之一。保留工资决定是否接受offer → 直接影响失业持续期；搜索强度决定offer到达率。
- **提升理由**：保留工资(rw2)和搜索小时数(js7)是失业持续期和就业转移概率的直接决定因素。相比Inflation Belief（通过消费→产出→就业的间接链），本维度对失业率预测更具第一性。
- **题项支持**：✅ LM Quarterly 提供保留工资、灵活性指标、搜索强度、offer结果等完整信息。
- **已知局限**：LM Quarterly 覆盖期 2013-2021，频率为季度。模型中需要月度更新时，需要插值或假设月内恒定。
- **分布构造**：✅ reservation_wage_ratio + search_hours + flexibility → PCA or index，合理。
- **时间属性**：✅ 中速状态变量。

### 异质性 5：Housing Mobility Friction（MVP）

**审计结论：保留，无重大修改。**

- **名称准确性**：✅ 准确。
- **与主目标一致性**：✅ 通过地理流动性 → 就业匹配效率 → 失业持续期的通道影响劳动市场。
- **机制解释充分性**：✅ lock-in效应、搬迁成本、杠杆放大均有支撑。
- **题项支持**：✅ Core Q41-43（持有状态）+ Housing HQH5/6（房贷/搬迁障碍）完整。
- **建议增强**：对于失业率预测，关键变量是"是否因住房锁定而无法迁移到就业机会更好的地区"。建议在构造时增加 HQH6（搬迁障碍）与 LM 地理搜索范围的交叉。
- **分布构造**：✅ Owner-Low-LTV / Owner-High-LTV / Renter-Stable / Renter-Mobile 四类合理。
- **时间属性**：✅ 慢变量。

### 异质性 6：Consumption Adjustment Rule（MVP）

**审计结论：保留，无重大修改。**

- **名称准确性**：✅ 准确。
- **与主目标一致性**：✅ MPC异质性影响总需求，总需求变化反馈到劳动需求和就业。同时，消费调整规则也反映agent在失业冲击下的缓冲策略选择。
- **机制解释充分性**：✅ 充分。
- **题项支持**：✅ Spending qsp12n（正向MPC）和 qsp13new（负向MPC）直接测量MPC。
- **建议增强**：构造 asymmetry_ratio = qsp13new / qsp12n 以捕捉损失厌恶程度。
- **分布构造**：✅ latent class (Saver/Spender/Smoother) 或 rule-based 分类合理。
- **时间属性**：✅ 慢变量。


### 异质性 7：Inflation Belief Heterogeneity（扩展层，已从MVP降级）

**审计结论：保留在第二阶段，降级合理。**

- **名称准确性**：✅ 准确。
- **与主目标一致性**：⚠️ 间接相关。传导链：通胀预期 → 工资谈判/消费时机 → 产出 → 劳动需求 → 就业。链条较长，不是失业率的一阶驱动力。
- **降级理由**：除非模型显式包含价格/工资设定模块（第一版不包含），否则通胀预期无法直接进入失业率的决定方程。在第二阶段加入工资谈判模块后，再引入本维度更合理。
- **题项支持**：✅ Q9系列提供完整密度预测，数据质量高。
- **分布构造**：✅ Q9_cent50 + Q9_iqr 二维分布，合理。
- **时间属性**：✅ 快速预期变量。

### 异质性 8：Credit Access Constraint（扩展层）

**审计结论：保留在第二阶段。**

- **名称准确性**：✅ 准确。
- **与主目标一致性**：⚠️ 间接相关。信贷约束 → 消费平滑能力 → 搜索持续能力。但与 Liquidity Fragility 有功能重叠。
- **与Liquidity Fragility的区别**：流动性是"手上有没有现金缓冲"，信贷是"能不能从市场上借到钱"。理论上独立，但在MVP阶段可以先用流动性近似。
- **关于 discouraged borrower (N11)**：概念上非常有价值——它是一个内生决策节点，在第二阶段 borrowing block 扩展时应建模为决策分支。
- **题项支持**：✅ Credit Access Survey N1-N11 完整覆盖。
- **时间属性**：✅ 中速状态变量。

### 异质性 9：Numeracy & Planning（扩展层）

**审计结论：保留在第二阶段。**

- **名称准确性**：✅ 准确。
- **与主目标一致性**：⚠️ 间接。影响预期形成质量和规划能力，但不直接决定就业转移。
- **MVP阶段替代方案**：教育水平（Q36）可作为粗糙代理变量，已包含在人口学背景中。
- **题项支持**：✅ QNUM1-9 提供标准化算术测试。
- **时间属性**：✅ 慢变量。

### 异质性 10：Risk Preference（扩展层）

**审计结论：保留在第二阶段。**

- **名称准确性**：✅ 准确。
- **与主目标一致性**：⚠️ 间接。影响自雇/创业决策和工作类型选择，但非失业率主通道。
- **题项支持**：✅ QRA1/QRA2 提供标准化风险态度量表。
- **时间属性**：✅ 慢变量。

### 异质性 11：Balance Sheet Composition（校准/验证层）

**审计结论：保留在校准层。**

- **名称准确性**：✅ 准确。
- **定位理由**：HH Finance 仅覆盖 2014-2019 且已停更。适合用于初始人口的财富分布校准，不适合动态更新。
- **时间属性**：✅ 慢变量。

### 异质性 12：Policy Sensitivity（校准/验证层）

**审计结论：保留在校准层。**

## 四、审计结论

经审计，当前12个异质性维度**全部保留**，无需删除任何维度。主要调整为：

1. **两个层级变动**：Labor Search Friction 升入 MVP；Inflation Belief 降至扩展层
2. **两个构造修正**：Liquidity Fragility 数据主干调整；Labor Fragility PCA方向修正
3. **一个补全任务**：Labor Fragility 正文结构补全

审计通过，进入边界冻结。
